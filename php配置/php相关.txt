php5.3已经不再支持zendOptimizer，所以不能安装．可以用zend-guard取代
Zend dropped support for 5.3, they have a new project now called zend guard. I believe it requires a key to use
# yum search zend
# yum info php-zend-guard-loader.i386
# yum install php-zend-guard-loader.i386

php5.2以下安装ZendOptimizer
# wget http://downloads.zend.com/optimizer/3.3.3/ZendOptimizer-3.3.3-linux-glibc23-i386.tar.gz
# tar -xzvf ZendOptimizer-3.3.3-linux-glibc23-i386.tar.gz
# ./ZendOptimizer-3.3.3-linux-glibc23-i386/install.sh
基本上一路回车就可以了，安装会自动检测php.ini和apache的路径。
